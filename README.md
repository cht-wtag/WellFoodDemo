# README

## WellFoodDemo

* Ruby version
```bash
 ruby '2.6.5'
```

* Rails version
```bash
 gem 'rails', '~> 6.0.1'
```

* Database
```bash
 latest postgres
```

** To run the project

- Clone the repository

  ```bash
    https://github.com/cht-wtag/WellFoodDemo.git
  ```

- Run command

  ```bash
    gem install bundler
    bundle install
    rails db:create
    rails db:migrate
    rails db:seed
    rails s
  ```
